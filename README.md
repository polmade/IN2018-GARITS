# IN2018-GARITS

Repo for second year IN2018 module at CUoL

Team 7


## Getting Started

Clone the project and open it in IntelliJ. Click run.

## Prerequisites

JDBC driver MariaDB version 3.0.4 needed.
Use dump file in ../GATIS/data to build database.
